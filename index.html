使用說明 & GAS 後端部署教學

1 建立 Google 試算表

請至 Google Drive 雲端硬碟，新建一個全新的「試算表 (Google Sheet)」。不需預先建立任何工作表或欄位，本系統具備自動化初始化功能。

2 貼入 Google Apps Script 程式碼

在試算表中，點選上方選單「擴充功能」 -> 「Apps Script」。將編輯器內預設的所有程式碼清空，並貼上下方完整的 GAS 程式碼。

// 正在讀取 GAS 原始碼...

3 網頁應用程式部署 (Web App)

  • 在 Apps Script 專案右上角點擊「部署」 -> 「新增部署」。
  • 點擊齒輪圖示，選取「網頁應用程式」類型。
  • 在「執行身分」中選擇：「我自己」 (Me)。
  • 在「誰可以存取」中選擇：「任何人」 (Anyone) (這樣前端靜態網頁才能呼叫 API)。
  • 點選「部署」按鈕。Google 將要求您進行帳號權限授權，請完全同意授權。
  • 部署完成後，複製畫面上產生的「網頁應用程式網址 (Web App URL)」。

4 貼回本系統並儲存

回到本平台，點擊右上角的設定 (⚙️) 按鈕。將剛才複製的 Web App 網址貼入「GAS API 部署網址」欄位中，然後點擊儲存，即可啟用。
